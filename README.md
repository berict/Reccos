# Reccos

## Database

### Global Filters
* Boot types (FG / AG / SG / HG / TF / IN)
* Boot class (Limited / Elite / Pro / Club / Team / Intro)
* Size
* Boot user (Men / Women / Kids)
* Condition (Used / New / New without box)
* Brand
* Selling format
